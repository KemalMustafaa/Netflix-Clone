# Netflix-Clone
Ini adalah proyek Netflix UI Clone sederhana yang saya bangun sebagai bagian dari proses pembelajaran saya dalam menguasai fundamental pengembangan web front-end. Proyek ini memfokuskan pada replikasi desain antarmuka pengguna (UI) Netflix yang ikonik untuk melatih ketajaman visual dan kemampuan coding saya.
